## 📖 简介

公元 2147 年，记忆熵值异常攀升，人类面临记忆奇点危机。记忆熵研究项目首席科学家林默发现，神秘信息来自已故十五年的女儿小雅。随着逆熵协议启动，一个惊天的真相逐渐浮出水面……

这是一部关于记忆、身份、人性和进化的硬科幻作品，融合了刘慈欣式的宏大叙事与深刻的人性探讨。

---

## 📚 卷目

### 第一卷：观测者（第 1-5 章）
记忆熵危机初现，神秘信息来自已故女儿，逆熵协议启动

### 第二卷：追溯者（第 6-10 章）
记忆碎片揭示真相，十五年前的事故另有隐情，叛徒现身

### 第三卷：反抗者（第 11-15 章）
记忆抵抗组织浮出水面，加速器秘密，维度之战

### 第四卷：升维者（第 16-20 章）
最终对决，记忆奇点降临，人类的选择，新的开始

## 🎯 核心设定

### 记忆熵
记忆作为可量化的物理实体，信息在时空中留下的熵增痕迹。

### 记忆奇点
熵值达到 100% 时的临界点，所有记忆转化为纯能量。

### 逆熵协议
逆转局部时空熵增的协议，可降低熵值但会抹除记忆。

### 纯信息生命
人类进化的下一阶段，脱离肉体的纯信息存在。

### 维度升维
从三维空间向更高维度的转换。
